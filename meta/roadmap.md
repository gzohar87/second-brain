# Roadmap

## Phase 1: Foundation (done)

Directory structure, CLAUDE.md, basic wiki layout, and initial tooling scripts.

## Phase 2: Skills (done)

Ingest, query, and lint skills extracted into standalone skill files.

## Phase 3: Meta-wiki buildout (in progress)

Internal documentation of the framework itself — architecture, decisions, concepts, and this roadmap.

## Phase 4: Bootstrap script + docs

A single command to scaffold the framework into a new or existing repo, plus user-facing documentation.

## Future

- Search integration (qmd) for full-text retrieval across wiki pages
- Marp slide generation from wiki content
- Dataview-style frontmatter queries for dynamic indexes
